# 🧮 Calculadora GUI

Calculadora de mesa simples, com interface gráfica construída em **Python + Tkinter** (biblioteca padrão, sem dependências externas).

## ✨ Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão
- Suporte a atalhos de teclado (números, operadores, `Enter` para calcular, `Esc` para limpar, `Backspace` para apagar)
- Tratamento de erros (ex.: divisão por zero) sem travar a aplicação
- Interface escura, limpa e responsiva ao clique

## 🖥️ Pré-requisitos

- Python 3.8+
- Tkinter (já incluso na instalação padrão do Python na maioria dos sistemas)

## ▶️ Como executar

```bash
python app.py
```
